# ckb-dao-vote

See [specification](./docs/ckb-dao-vote.md).

## Requirement
Install clang-19:
```bash
wget https://apt.llvm.org/llvm.sh
chmod +x llvm.sh
sudo ./llvm.sh 19
```

## How to Build and Test

```
make build
make test
```
